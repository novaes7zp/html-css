Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito ussando a tag "B" (não semântica).

Nesta frase, tenho um termo em destaque usando tag STRONG (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em italico usando tag I (não semântica).

Nesta frase, temos um termo em Ênfase usando a tag EM (semântica).

Texto marcado

Podemos criar tambem um texto marcado usando a tag MARK.

Podemos criar a tag MARk quantas vezes que quiser.

Texto grande e pequeno

Estamos criando um texto GRANDE e um texto pequeno.

Texto deletado

Podemos marcar um texto como excluido/riscado para indicar que o texto deve ser lido mas não considerado.

Texto inserido

Podemos marcar um txto como inserido para dar um enfase e indicar que ele foi adicionado depois.

Existe tambem o sublinado com a tag U (não semântica).

Texto sobrescrito

Para inserir coisa do tipo x25+69. usar a tag SUP significa para cima.

Para inserir coisa do tipo H20 usar a tag SUB significa para baixo.
